# Addition Calculator
A simple and efficient Python script that performs basic addition operations.

## 🚀 Features
* User-friendly command-line interface.
* Fast calculation of two numbers.
* Built using Python.
* safe for kids 
* made from the scratch so that its understood to the kids

## 🛠️ How to Run
1. Make sure you have **Python** installed on your computer.
2. Download the `addition calc.py` file.
3. Open your terminal or command prompt.
4. Run the script using:
   ```bash
   python "addition calc.py"
