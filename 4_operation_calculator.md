# 4 Operation Calculator 🧮

A clean and simple Python script that performs basic arithmetic operations. This project focuses on utilizing **conditional logic** and **formatted strings** to create a user-friendly terminal experience.

## 🚀 How it Works
The program follows a linear execution path:
1. **User Greeting:** Displays a clean header with available operations.
2. **Input Collection:** Prompts the user for two numbers (converted to `float` for decimal support) and an operator.
3. **Logic Chain:** - Uses `if` and `elif` to match the operator.
   - Uses **Nesting** within the division block to prevent mathematical errors.
4. **Output:** Returns the result using **F-strings** for clear formatting.
