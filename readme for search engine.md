# ✨ Word Search and Count Engine ✨

## 📝 Description
This is a Python-based utility developed during my **"Result Summer"** coding marathon. After finishing my 10th Standard Board Exams, I wanted to build a tool that demonstrates the core concepts of **File I/O (Input/Output)** and **String Manipulation**.

The engine allows a user to select any text file, "by-hearts" the content into memory, and performs a case-insensitive search to find exactly how many times a specific word appears.

## 🚀 Key Features
* **Dynamic File Selection:** Choose any `.txt` file in your directory.
* **Smart Search:** Uses `.lower()` logic to find matches regardless of capitalization (e.g., searching for "python" will find "Python").
* **Clean UI:** Uses emojis and formatted output for a better user experience.
* **Memory Efficient:** Properly opens and closes files to ensure no data leaks.

## 🛠️ How to Use
1.  **Clone the Repository:**
    Download `search engine.py` and the sample `data.txt` provided in this repo.
    
2.  **Setup:**
    Place both the script and the text file in the same folder.
    
3.  **Run the Engine:**
    Open your terminal or VS Code and run:
    ```bash
    python "search engine.py"
    ```
    
4.  **Execute Search:**
    * Enter the filename: `data.txt`
    * Enter the word you want to find (e.g., "Mumbai" or "Logic").
    * View your results!

## 📂 Project Structure
* `search engine.py` - The primary Python script containing the logic engine.
* `data.txt` - A sample 5-page text file for testing the search capabilities.
