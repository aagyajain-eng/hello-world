# AVERAGE CALCULATOR
A simple, efficient Python-based tool to calculate the arithmetic mean of a set of numbers. This project is part of the [hello-world](https://github.com/aagyajain-eng/hello-world) repository.

## 🚀 Features
* Supports manual input of TWO numerical values.
* Handles floating-point numbers for precise calculations.
* Clean and easy-to-read console output.
