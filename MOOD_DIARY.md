# ✨ Mood Diary ✨

## 📝 Introduction
Welcome to the **Mood Diary**! This is a Python-based digital journal designed to help users track their daily emotions and thoughts. It uses a structured data approach to save entries, making it easy to reflect on your progress over time.

> "A small script to track big feelings." 🚀

---

## ⚙️ How It Works
The script utilizes core Python concepts to manage data:
* **Variables & Inputs:** Captures dates, mood scores, and personal notes.
* **File Handling:** Uses `open()` in `a` (append) mode to save data without deleting past history.
* **Logic Blocks:** Uses `if-elif-else` statements to create a "Hub" where you can choose between writing or reading.
* **Data Formatting:** Automatically organizes entries using **Comma Separated Values (CSV)** principles.

---

## 🚀 How to Use It
Follow these simple steps to start your diary:

1. **Run the Script:** Start the program in your Python environment.
2. **Choose Your Mode:**
   * Type `W` to **Write** a new entry.
   * Type `R` to **Read** all your past entries.
3. **Log Your Data:** Enter the date, a score from 1-10, and a quick note about your day.
4. **View Progress:** Select the read mode at any time to see your full history printed in the console.

---

## 📊 Data Structure
The entries are saved in a file named `mood_diary.txt` (or `.csv`) in the following format:
`Date, Mood Score, Note`

Example:
`08/04, 9, Successfully finished my first database script! 🥳`
